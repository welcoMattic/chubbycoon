# Chubbycoon blog

## Requirements

```
$ pip install --user -r requirements.txt
```

## Installation

```
$ fab start
```

## Build js/css stuff

```
$ fab webpack_prod
```

To watch, for dev

```
$ fab webpack_watch
```
